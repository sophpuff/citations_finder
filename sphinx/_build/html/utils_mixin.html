get_doi_from_path(path)[source]

Extracts the DOI from a given file path by removing “.pdf” and replace “_” with “/” to conform with the DOI format.

Example: Convert “10.11646_phytotaxa.388.2.3.pdf” to “10.11646/phytotaxa.388.2.3”

The path of the file containing the DOI
