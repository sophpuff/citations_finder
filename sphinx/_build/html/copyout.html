write_match(cur_match, filehandle, db)[source]

Write a matched data record to a file.

The matached data record has 2 parts. The first part includes DOI. The second part is “identifier” from “matched_collectin_ids” table in the databse (example: cas 229582)

Parameters:

  • cur_match (tuple) – A tuple containing matched data fields, including DOI, collection, origin_path,published_date, journal_title, date_added, notes, and digital_only.

  • filehandle (file) – The file handle where the matched data will be written.

  • db (DoiDatabase) – An instance of the DoiDatabase class used for retrieving DOI record information.
