scan_pdfs(start_year, end_year, rescore=False, directory='./')[source]

Scans PDFs for DOIs within the specified year range. It retrieves DOIs that have been downloaded but not yet scanned.

Parameters:

  • start_year (int) – The starting year of the range for which PDFs should be scanned.

  • end_year (int) – The ending year of the range for which PDFs should be scanned.

  • rescore (bool, optional) – Flag indicating whether to rescore previously scanned DOIs, defaults to False.

  • directory (str, optional) – The directory where the scanned PDFs should be saved, defaults to “./”.
